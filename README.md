# 👋 Willkommen bei PhoenixWP

**PhoenixWP** ist eine unabhängige WordPress‑Plugin‑Schmiede aus Deutschland.  
Ich entwickle klare, zuverlässige und praxisorientierte Plugins für WooCommerce und WordPress – mit Fokus auf:

- 🧩 B2B‑Funktionen & Preislogik  
- 🛒 WooCommerce‑Optimierungen  
- 🖼️ Medien‑SEO & Alt‑Tag‑Automatisierung  
- 🧱 saubere, wartbare und update‑sichere Architektur  

Mein Ziel ist es, Werkzeuge zu bauen, die echte Probleme lösen – minimalistisch, performant und ohne unnötigen Ballast.
